# 📊 Statistics for AI — Machine Learning Repository

> A structured collection of machine learning concepts, statistical foundations, supervised & unsupervised learning notebooks, and classification models — all in one place.

## 🗂️ Repository Structure

statistics-for-AI/
│
├── ml_foundation/                          # Core ML concepts & fundamentals
├── statistics/                             # Statistical methods for AI/ML
├── supervised_learning/                    # Supervised learning algorithms
├── supervised_learning_classification.ipynb  # Classification notebook
├── unsupervised_learning/                  # Unsupervised learning algorithms
└── README.md

## 📚 Topics Covered

### 🔢 1. ML Foundation (`ml_foundation/`)
- Introduction to Machine Learning
- Types of ML: Supervised, Unsupervised, Reinforcement Learning
- Bias-Variance Tradeoff
- Overfitting & Underfitting
- Train / Validation / Test Split
- Cross-Validation
- Feature Engineering & Feature Selection
- Model Evaluation Metrics

---

### 📐 2. Statistics (`statistics/`)
- Descriptive Statistics (Mean, Median, Mode, Variance, Std Dev)
- Probability Theory & Distributions
  - Normal Distribution
  - Binomial Distribution
  - Poisson Distribution
- Hypothesis Testing (t-test, chi-square, ANOVA)
- Correlation & Covariance
- Bayes' Theorem
- Central Limit Theorem
- Confidence Intervals & p-values



### 🎯 3. Supervised Learning (supervised_learning)

#### Regression
- Linear Regression
- Polynomial Regression
- Ridge & Lasso Regression
- Gradient Descent

#### Classification
- Logistic Regression
- Decision Trees
- Random Forest
- Support Vector Machines (SVM)
- K-Nearest Neighbors (KNN)
- Naive Bayes

#### Model Evaluation
- Confusion Matrix
- Accuracy, Precision, Recall, F1-Score
- ROC Curve & AUC
- Cross-Validation



### 🔍 4. Supervised Learning — Classification Notebook
**File:** `supervised_learning_classification.ipynb`

An end-to-end classification notebook covering:
- Data Preprocessing & EDA
- Feature Engineering
- Model Training (multiple classifiers)
- Hyperparameter Tuning
- Performance Comparison



### 🔵 5. Unsupervised Learning (`unsupervised_learning/`)

#### Clustering
- K-Means Clustering
- Hierarchical Clustering
- DBSCAN

#### Dimensionality Reduction
- Principal Component Analysis (PCA)
- t-SNE
- LDA (Linear Discriminant Analysis)

#### Association
- Apriori Algorithm
- FP-Growth

## 🚀 Getting Started

### 1. Clone the Repository

git clone https://github.com/Amrita/statistics-for-AI.git
cd statistics-for-AI


### 2. Install Dependencies

pip install numpy pandas matplotlib seaborn scikit-learn jupyter


### 3. Launch Jupyter Notebook

jupyter notebook


## 📈 Learning Path (Recommended Order)


1. ml_foundation/        →  Understand core ML concepts
2. statistics/           →  Build statistical intuition
3. supervised_learning/  →  Learn regression & classification
4. supervised_learning_classification.ipynb  →  Hands-on practice
5. unsupervised_learning/  →  Clustering & dimensionality reduction
